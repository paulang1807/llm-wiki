# Databases

Core data storage and retrieval concepts, covering relational databases (SQL), NoSQL, and database integration patterns.

## Relational Databases (SQL)

Relational database systems organize data into tables with predefined schemas.

- **SQLite**: A lightweight, serverless database engine integrated with Python.
- **SQLAlchemy**: A powerful toolkit and ORM for SQL in Python.
